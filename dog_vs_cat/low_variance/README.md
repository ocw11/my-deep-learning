在high_variance中的模型表现不是很理想，因为模型出现了过拟合现象，在low_variance_model中，我们加入了正则化来防止过拟合。
解决过拟合的方法如下：
（1）数据增强
（2）增加Dropout层
（3）增加正则化项
（4）降低网络复杂度
（5）早停（early stopping）
