	我们之前构建了两种网络：high_variance_model和low_variance_model，前者能很好地分类出训练集，但对于验证集的表现不佳，出现过拟合现象；而后者虽然相较于前者，对于验证集的表现较好，但仍达不到很高的预测准确度，这是因为我们所使用的数据较少，只靠这些数据来从头开始训练自己的神经网络会很困难，所以想要进一步提高准确率，我们会使用预训练网络模型。
	预训练网络是保存好的网络，已经进过大量的数据训练。本次我们采用VGG16架构，这是在2014年开发的，虽然这是个非常老的网络架构，目前很多模型都远超VGG16，但是VGG16和我们所建立的网络非常相似，对于新手比较好理解，所以采用它。
	使用预训练网络有两种方法：特征提取（feature extraction）和微调模型（fine-tuning）。
